# SALA — Smart Adaptive Learning Assistant

An AI-powered adaptive learning platform designed to personalize education using intelligent recommendations, accessibility-focused learning, interactive quizzes, visual learning systems, and teacher analytics.

---

# 🚀 Overview

SALA (Smart Adaptive Learning Assistant) is a modern educational platform built to improve student learning experiences through AI-driven personalization and adaptive educational workflows.

The platform dynamically recommends learning content, tracks student performance, provides visual learning support, and enables teachers to monitor classroom analytics through an interactive dashboard.

This project was developed as a hackathon team project focused on solving modern educational challenges using AI and accessible technology.

# 🧠 How SALA Works

## Student Workflow
1. User logs into the platform
2. AI recommends learning modules
3. Student studies visual content
4. Student attempts quizzes
5. Performance is analyzed
6. Adaptive recommendations improve learning flow

---

## Teacher Workflow
1. Teacher accesses dashboard
2. Student performance data is analyzed
3. Quiz statistics are monitored
4. Learning patterns are observed
5. Progress tracking improves teaching decisions

# ⚙️ Installation & Setup

## Clone Repository

```bash
git clone https://github.com/nandakishore7054/sala-ai-learning-platform.git
```

---

## Navigate to Project

```bash
cd sala-ai-learning-platform
```

---

## Install Dependencies

```bash
npm install
```

---

## Run Development Server

```bash
npm run dev
```

---

# 🔐 Environment Variables

Create a `.env` file and configure:

```env
VITE_FIREBASE_API_KEY=your_key
VITE_FIREBASE_AUTH_DOMAIN=your_domain
VITE_FIREBASE_PROJECT_ID=your_project
VITE_FIREBASE_STORAGE_BUCKET=your_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
VITE_GEMINI_API_KEY=your_gemini_key
```
